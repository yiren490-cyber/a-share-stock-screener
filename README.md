# A股实时筛选器

这是一个零构建的本地网页应用，用免费公开行情源先跑通 A 股实时数据、基础筛选和点击查看主副图。

## 使用方式

启动本地服务后，用浏览器访问：

- 筛选器首页：`http://127.0.0.1:8787/`
- 实时盯盘页：`http://127.0.0.1:8787/watch.html`

当前电脑的局域网地址是 `http://10.1.0.14:8787/`，同一网络里的其他设备也可以尝试用这个地址打开。

如果服务没有启动，可以运行：

```powershell
& 'C:\Users\Administrator\AppData\Local\Programs\nodejs-portable\node-v24.14.0-win-x64\node.exe' server.js
```

筛选器页面右上角有“实时盯盘”入口，实时盯盘页也可以返回筛选器。

## 当前功能

- 拉取 A 股实时行情，当前使用腾讯行情接口。
- 使用内置示范股票池，覆盖沪市主板、深市主板、创业板、科创板、北交所。当前只有 20 个是因为还没有接入完整 A 股股票库。
- 按股票名称/代码搜索。
- 股票类型使用下拉弹窗多选，已选类型会在输入框里用逗号展示。
- 按上市天数、流通市值、涨跌幅、成交额、换手率、量比筛选。
- 点击表头排序。
- 导出筛选结果 CSV。
- 点击股票后显示日 K 主图和副图。
- 通过独立的实时盯盘页查看分时图、盘口、4 个周期 K 线和预警音频。
- 主图均线周期可以勾选。
- 图表支持放大、缩小、十字光标和悬浮查看日期、价格、成交量及指标值。
- 副图拆分为成交量、MACD、主力状态，并可设置开启 1 到 5 个副图。

## 数据源说明

第一版使用腾讯公开行情接口。公开接口适合原型验证，但稳定性和授权边界不等同于正式授权数据源。后续可以把数据源替换为付费或授权接口，并接入完整全市场股票库。

主力状态指标已接入对应市场指数收盘价作为 `INDEXC` 参与计算，但和通达信仍可能因为复权方式、成交量单位、`CAPITAL` 口径不同而有差异。
